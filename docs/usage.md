# Web Interface

Search provides a clean, privacy-focused web interface for searching the web.

## Basic Search

1. Navigate to your Search instance
2. Enter your search query in the search box
3. Press Enter or click the search button

## Search Categories

Switch between different search categories using the tabs:

- **General**: Web search results
- **Images**: Image search
- **Videos**: Video search
- **News**: News articles
- **Maps**: Map results

## Bang Commands

Bang commands are shortcuts that let you search other sites directly:

| Bang | Site |
|------|------|
| `!g` | Google |
| `!ddg` | DuckDuckGo |
| `!b` | Bing |
| `!w` | Wikipedia |
| `!yt` | YouTube |
| `!gh` | GitHub |
| `!so` | Stack Overflow |
| `!r` | Reddit |
| `!tw` | Twitter |
| `!amz` | Amazon |

### Usage

```
!w privacy
```

This will redirect you to Wikipedia's search for "privacy".

## Search Operators

Use operators to refine your search:

| Operator | Example | Description |
|----------|---------|-------------|
| `"..."` | `"exact phrase"` | Search for exact phrase |
| `-` | `privacy -google` | Exclude term |
| `site:` | `site:github.com` | Search specific site |
| `filetype:` | `filetype:pdf` | Search file type |

## Settings

Click the settings icon to customize your search experience:

### Display

- **Theme**: Light or Dark (Dracula)
- **Results per page**: 10, 20, 50, or 100
- **Open in new tab**: Open results in new tabs

### Privacy

- **Safe Search**: Off, Moderate, or Strict
- **Image Proxy**: Enable/disable image proxying
- **Autocomplete**: Enable/disable search suggestions

### Region

- **Language**: Preferred language for results
- **Region**: Preferred region for results

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `/` | Focus search box |
| `Enter` | Search |
| `↑/↓` | Navigate results |
| `Enter` | Open selected result |
| `Tab` | Switch categories |
| `Esc` | Clear search |
| `?` | Show shortcuts |

## Privacy Features

### No Tracking

- No cookies for tracking
- No search history stored
- No personal data collection

### Image Proxy

Images are proxied through your Search instance to prevent third-party tracking.

### Tor Support

Access your Search instance via Tor for additional privacy.

## Mobile Usage

Search is fully responsive and works on mobile devices:

- Swipe left/right to switch categories
- Pull down to refresh
- Use the share button to share searches
